== Description ==

== Installation ==

This section describes how to install the plugin and get it working.

1. Install through your wordpress plugin panel or Upload `prwirepro.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Open the Pr Wire Pro dashboard from within your wordpress panel.
4. Fill our the form and submit unlimited a press releases to the top 5 free pr sites. Form is submitted directly to Light Image Media. After review we submit the releases for free on your behalf.
5. Submit to 25 + top pr networks for wider distribution - Visit https://lightimagemedia.com/press to learn more about our premium submission service.
